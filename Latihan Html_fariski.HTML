Apa Itu HTML?

HTML itu adalah sebuah bahasa yang menggunakan markup atau penanda untuk membuat halaman web. penanda atau markup ini, nanti akan kita sebut dengan Tag. Penanda atau markup ini, nanti akan kita sebut dengan HTML berperan untuk menentukan struktur konten dan tampilan dari sebuah Web. Kalau kita ibaratkan nih.HTML itu seperti batu bata untuk membangun rumah. Batu bata ini dapat di susun, sehingga menjadi fondasi dasar. HTML kelebihan seperti memiliki sumber komunitas yang cukup besar dengan penggunaan yang luas. Tidak hanya Itu, kelebihan lainnya adalah HTML mudah untuk di pelajari oleh seorang Web Developer Pemula. kelebihan lainnya adalah HTML merupakan bahasa pemograman yang gratis atau open source
